## Task ID: 2 - Калькулятор материалов каркасного дома
### Work Task
Создать полноценное веб-приложение для расчёта материалов каркасного дома по технологии Frame House (канадская технология).

### Work Summary

Создано полнофункциональное веб-приложение с полным расчётом материалов для строительства каркасного дома.

#### Структура созданных файлов:

**1. Типы данных (`src/lib/types.ts`):**
- HouseParams - все входные параметры дома
- MaterialItem - структура материала с количеством и единицами измерения
- CalculationResult - результат расчёта с материалами и сводкой
- CALCULATION_CONSTANTS - константы для расчётов (размеры ОСП, шаги, коэффициенты)

**2. Библиотеки расчётов (`src/lib/calculations/`):**

- **foundation.ts** - расчёт фундамента:
  - Свайный: количество свай, бетон, арматура, ростверк, оголовки
  - Ленточный: бетон, арматура, опалубка, гидроизоляция, песчаная подушка
  - Плитный: бетон, двухслойная арматурная сетка, гидроизоляция, песок и щебень

- **frame.ts** - расчёт каркаса:
  - Стойки внешних стен с учётом проёмов
  - Нижняя и верхняя обвязка
  - Лаги пола и перекрытия
  - Стропильная система (расчёт длины стропил по теореме Пифагора)
  - Обрешётка и контробрешётка
  - Металлические уголки и крепёж

- **sheathing.ts** - расчёт обшивки:
  - ОСП для внешних и внутренних стен
  - Ветрозащитная мембрана
  - Пароизоляция
  - ОСП для крыши (под мягкую кровлю)

- **insulation.ts** - расчёт утепления:
  - Утеплитель для стен, пола, потолка
  - Звукоизоляция для внутренних перегородок
  - Утепление мансарды
  - Утепление цоколя

- **fasteners.ts** - расчёт крепежа:
  - Гвозди строительные
  - Саморезы для ОСП, ГКЛ, сайдинга
  - Металлические уголки и пластины
  - Анкеры и дюбели
  - Крепёж для окон и дверей (пена, ПСУЛ, отливы)

- **roof.ts** - расчёт кровли:
  - Кровельный материал (металлочерепица, мягкая кровля, профнастил)
  - Подкровельная гидроизоляция
  - Конёк, карнизные и торцевые планки
  - Ендовы (для вальмовой крыши)
  - Водосточная система
  - Снегозадержатели и вентиляция

- **index.ts** - объединяющий модуль:
  - calculateAll() - вызывает все расчёты и формирует итоговый результат
  - calculateFinish() - внешняя и внутренняя отделка
  - calculateWindowsDoors() - окна и двери
  - calculateSummary() - сводка (объём пиломатериалов, утеплителя, площади)

**3. API endpoint (`src/app/api/calculate/route.ts`):**
- POST - принимает параметры дома, валидирует и возвращает расчёт
- GET - документация API

**4. UI (`src/app/page.tsx`):**
- Красивый заголовок с описанием и бейджами
- Форма с 6 вкладками:
  - Основные параметры (длина, ширина, этажность, мансарда)
  - Крыша (тип, угол наклона, материал)
  - Стены (количество, толщина, шаг стоек)
  - Окна и двери (малые, средние, большие окна, двери)
  - Фундамент (тип, глубина)
  - Дополнительные опции (утеплитель, отделка)
- Кнопка расчёта с индикацией загрузки
- Результаты расчёта:
  - Сводная информация (5 ключевых показателей с иконками)
  - Таблицы материалов по категориям (8 категорий)
  - Общее количество позиций

#### Особенности реализации:
- Все расчёты основаны на СП 31-105-2002
- Шаг стоек по умолчанию 600 мм
- Коэффициент запаса 10% на отходы
- Коэффициент нахлёста 15% для мембран
- Длина стропил рассчитывается по теореме Пифагора с учётом угла наклона
- Адаптивный дизайн для мобильных устройств
- Цветовая схема: оранжевый/синий акценты (строительная тематика)

#### Проверка качества:
- ESLint: ошибок нет
- Dev сервер: работает корректно
- Страница загружается без ошибок
